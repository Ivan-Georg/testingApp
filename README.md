# testingApp


Jag har gjort ett program som automatiskt testar en mockad databas. Jag använde mig av en UserManager-klass och ett antal testfall. Dessa är bland annat AddUser, GetUser och RemoveUser. 

Det som jag har gjort väl i den här uppgiften är att jag delade upp koden i fliken för ökad tydlighet. Jag lyssnade på din synpunkt om att skriva kommenterar på engelska. 

Det som var svårt var att lära sig nya Framework och hanteringen av Moq och NUnit. Det tog ett tag att sätta sig in i vad det är och hur det fungerar. 

Automatisk testning verkar dock bra att kunna eftersom all kod och program måste testas. Automatisk testning är ett bra komplement till manuell testning, särskilt i stora progarm så att inte hela behöver köras.
